# Sublime Text 3 Snippets

## Installation

### Package Control

1. `ctrl+shift+p` &rarr; `Package Control: Add Repository` &rarr; `https://github.com/phts/sublime-snippets`
2. `ctrl+shift+p` &rarr; `Package Control: Install Package` &rarr; `sublime-snippets`
