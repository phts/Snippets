# Snippets

## Install

### Package Control

1. Open Command Palette &rarr; `Package Control: Add Repository` &rarr; `https://github.com/phts/Snippets`
2. Open Command Palette &rarr; `Package Control: Install Package` &rarr; `Snippets`
